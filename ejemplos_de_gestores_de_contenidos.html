Ejemplos de gestores de contenidos

«WordPress»

Es uno de los gestores de contenido más populares. Atiende a miles de usuarios y sitios web gracias a su flexibilidad de herramientas y su interfaz fácil de utilizar.

Aunque carece de herramientas básicas de creación web, tiene complementos útiles para crear rápidamente sitios funcionales de comercio electrónico.

Sus herramientas de aprobación de contenido, compatibilidad con SSL¹ y privilegios granulares son los encargados de proteger el contenido. Limitando el acceso según los niveles administrativos que se hayan establecido. WordPress tiene verificación de correo electrónico, características de notificación automática de problemas y herramientas de captcha² para mantener el sitio seguro para el propio usuario.

Además, tiene una gran cantidad de plugins y complementos nativos o de terceros. Por tanto:

- Es muy sencillo de utilizar.
- Incluye varias plantillas.
- La personalización es muy completa.
- Muchos gestores de alojamiento web lo ofrecen instalado dentro de su servicio.

Existen dos opciones:

- wordpress.com: posee menos funcionalidades, ya que está orientada hacia principiantes.

- wordpress.org: posee múltiples funcionalidades para generar un sitio web profesional. 

      

«Magento»

Está orientado a crear atractivos y potentes sitios web para e-commerces. Para ello, ofrece soluciones con temas ya preparados y tiene una capacidad ilimitada para personalizar el sitio con un código propio. Además, es fácil de navegar en él, gracias a que su tablero de mandos tiene un aspecto y organización familiar a la mayoría de estos CMS.

Se pueden elegir entre plantillas ya creadas o crear unas propias. 

El sistema proporciona características propias de seguridad y privacidad para cumplir con las leyes vigentes y los requisitos de la industria de tarjetas de pago.

Al estar orientado hacia el e-commerce, cuenta con diversas funciones para la creación de catálogos de productos, además de herramientas para la fijación de precios.    

«Drupal»

Está destinado a ayudar a diseñadores profesionales a crear sitios web potentes. Espacios capaces de manejar grandes volúmenes de visitantes y cientos de páginas de contenido. Por tanto, a pesar de la que cuenta con complementos y plugins, no es fácil de usar por parte de diseñadores web novatos. 

Cuenta con diversos temas. Sin embargo, requiere de una gran cantidad de codificación para convertir ese sitio en algo realmente potente.

También incluye herramientas de blog, formularios de e-mail, motores de búsqueda y mapas del sitio. Además de una gama de add-ons³ orientada a los comercios y sitios web de negocios.

 

«Joomla»

Está diseñado para desarrolladores con experiencia en incorporar código. Posee una amplia selección de complementos de negocios, por lo que funciona bien para las empresas que buscan convertir su sitio web básico en algo más atractivo, único, interactivo y profesional.

Joomla posee casi todos los complementos de negocio gratuitos, aunque la selección de temas no es tan extensa.

      

  

«Prestashop»

Es otro de los gestores de contenido web más populares para el desarrollo de e-commerces. Posee gran cantidad de plantillas, tanto gratuitas como de pago.

* Algunas definiciones:

SSL

La sigla SSL significa Secure Sockets Layer (Capa de sockets seguros), un protocolo de seguridad que crea un enlace cifrado entre un servidor web y un navegador web.

Herramientas de captcha

Ayuda a proteger del spam y del descifrado de contraseñas pidiendo que se complete una simple prueba que demuestre que eres humano y no un ordenador que intenta acceder a una cuenta protegida con contraseña.

Add-ons

Del inglés extensión o añadidura, que puede referirse a: Expansión (videojuegos), una adición a un juego existente de rol (RPG), juego de mesa o videojuego. Complemento (Mozilla), una mejora instalable para los proyectos de la Fundación Mozilla.
